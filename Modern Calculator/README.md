# Modern Calculator 🧮

A sleek and responsive **web-based calculator** built with **HTML, TailwindCSS, and JavaScript**.  
Includes support for history tracking, keyboard shortcuts, dark mode, and smooth button animations.

---

## 🚀 Features
- ➕ Basic arithmetic operations (+, −, ×, ÷, %)
- 📝 **Calculation history** with ability to recall past results
- 🎨 **Responsive UI** using TailwindCSS
- 🌙 **Dark mode toggle** (saved in localStorage)
- ⌨️ **Keyboard support** for numbers, operators, backspace, and enter
- ⚡ Smooth button animations and hover effects
- 🗑️ Clear All (AC) and Clear History functions

---


---

## ▶️ How to Run
1. Clone or download this repository.
2. Open `calculator.html` in your browser.
3. Start calculating!

---

## 🎮 Keyboard Shortcuts
- **Numbers & Operators:** `0–9`, `+`, `-`, `*`, `/`, `%`, `.`
- **Enter / =** → Calculate result  
- **Backspace** → Delete last entry  
- **Esc** → Clear all  
- **History panel:** Toggle with history button
